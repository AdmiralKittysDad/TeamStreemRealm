================================================================================
                    PYRAMID REQUIREMENTS
================================================================================

BEACON MECHANICS:
Beacons require a pyramid of mineral blocks beneath them to activate.
Larger pyramids = more powerful effects and longer range.

PYRAMID TIERS:
┌─────────────────────────────────────────────────────────────────────────────┐
│ Tier │ Layers │ Base Size │ Blocks Needed │ Range  │ Effects Available      │
├─────────────────────────────────────────────────────────────────────────────┤
│  1   │   1    │    3×3    │      9        │ 20 blk │ Primary only           │
│  2   │   2    │    5×5    │    9+25=34    │ 30 blk │ Primary only           │
│  3   │   3    │    7×7    │  34+49=83     │ 40 blk │ Primary + Secondary    │
│  4   │   4    │    9×9    │ 83+81=164     │ 50 blk │ Primary + Secondary    │
└─────────────────────────────────────────────────────────────────────────────┘

RECOMMENDATION: 3-TIER PYRAMID (fits inside 7×7 tower base)
- Requires 83 mineral blocks per beacon
- Provides 40-block effect range
- Allows both primary AND secondary effects
- Fits perfectly within tower footprint

================================================================================
                    PYRAMID MATERIAL OPTIONS
================================================================================

Valid pyramid materials (can be mixed):
├── Iron Blocks (most common, easiest to obtain)
├── Gold Blocks (moderate difficulty)
├── Emerald Blocks (trading-based)
├── Diamond Blocks (expensive but beautiful)
└── Netherite Blocks (ultimate flex)

RECOMMENDED APPROACH:
Use IRON BLOCKS for practicality:
- 83 blocks × 2 towers = 166 iron blocks
- 166 blocks × 9 ingots = 1,494 iron ingots
- Obtainable through iron farms or mining

AESTHETIC OPTION:
Mix materials for visual interest (even though hidden):
- Layer 1 (bottom): Iron
- Layer 2 (middle): Gold
- Layer 3 (top): Diamond

                    CONSTRUCTION SEQUENCE
================================================================================

PHASE 1: Clear tower base area
├── Mark 7×7 footprint for each tower
├── Ensure Y:69 level is flat
└── Remove any water

PHASE 2: Build pyramid base (Y:69)
├── Place 49 iron blocks in 7×7 pattern
├── North tower at Z:-13 to -7
├── South tower at Z:+7 to +13
└── Verify alignment with platform

PHASE 3: Build pyramid middle (Y:70)
├── Place 25 iron blocks in 5×5 pattern (centered)
├── Begin tower walls around pyramid
├── Leave door opening on platform-facing side
└── Interior space for future stairs

PHASE 4: Build pyramid top (Y:71)
├── Place 9 iron blocks in 3×3 pattern (centered)
├── Continue tower walls
├── Interior should be accessible via door
└── Verify pyramid is complete

PHASE 5: Place beacons (Y:72)
├── Place beacon on top of pyramid center
├── Beacon should immediately activate (beam appears)
├── Beam will be WHITE until stained glass added
└── Do NOT configure effects yet

PHASE 6: Build tower (Y:73-89)
├── Continue tower construction as per Zone 4 blueprint
├── Ensure beacon beam has clear vertical path
├── No solid blocks above beacon until Y:90

PHASE 7: Add stained glass (Y:90)
├── Place colored stained glass above beacon
├── North: Cyan glass
├── South: Magenta glass
└── Beam should change color immediately

PHASE 8: Configure beacon effects
├── Right-click each beacon
├── Insert payment ingot
├── Select primary and secondary powers
├── Confirm configuration
└── Test effect by standing near tower

================================================================================
                    TROUBLESHOOTING
================================================================================

BEACON NOT ACTIVATING:
□ Check pyramid has valid mineral blocks (iron/gold/emerald/diamond/netherite)
□ Verify pyramid is directly under beacon (centered)
□ Ensure no blocks between beacon and sky (must see sky)
□ Confirm pyramid tiers are correct sizes

BEAM NOT COLORED:
□ Stained glass must be ABOVE beacon
□ Glass must be in beam path (centered)
□ Multiple glass blocks = blended color (use same color)
□ Ensure no other colored glass interfering

EFFECTS NOT WORKING:
□ Beacon must be configured (not just placed)
□ Payment ingot must be inserted
□ Powers must be selected and confirmed
□ Player must be within range (40 blocks for 3-tier)

BEAM BLOCKED:
□ Remove any solid blocks above beacon
□ Glass and water are OK (beam passes through)
□ Carpets, slabs, and some blocks block beams
□ Check entire column from Y:72 to Y:320
